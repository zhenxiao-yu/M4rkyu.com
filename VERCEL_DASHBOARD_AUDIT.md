## Naming Convention

Use short, portfolio-readable project names:

- Production identity: `m4rkyu-portfolio`
- Migration preview: `m4rkyu-next-preview` or `m4rkyu-2027-preview`
- Active apps: product names only, for example `politilens`, `m4rket-view`, `nimbus-storage`
- Archived experiments: prefix with `archive-`, for example `archive-client-vkzu`
- Avoid random suffixes in project names unless generated by Vercel aliases.
